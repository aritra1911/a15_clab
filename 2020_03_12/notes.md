# Functions
Syntax:
```
    return_data_type function_name(data_type parameter1, data_type parameter2, ..., data_typeparameter_n) { }
```
Example:
```c
int sum(int x, int y) {
    int z;
    z = x + y;
    return z;
}
```
But if you're a savvy programmer:
```c
int sum(int x, int y) {
    return x + y;
}
```
